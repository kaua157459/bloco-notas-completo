# 📝 Bloco de Notas Completo

Um editor de texto moderno e leve, feito em Python, com funcionalidades poderosas para quem escreve, programa ou só precisa de um bloco de notas eficiente!

## ✨ Funcionalidades

✅ Múltiplas abas  
✅ Salvar/Abrir arquivos `.txt`  
✅ Auto-completar palavras (com IA simples)  
✅ Numeração de linhas  
✅ Destaque de sintaxe para código Python  
✅ Temas claro e escuro  
✅ Reconhecimento de voz (ditado)  
✅ Leitura em voz alta com `pyttsx3`  
✅ Sistema de favoritos para abas  
✅ Formatação rica (negrito, itálico, sublinhado, cor de fonte, tamanho)  
✅ Menu de atalhos rápidos  
✅ Interface moderna com `ttk.Notebook`
